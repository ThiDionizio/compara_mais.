# compara_mais.

Aplicação de comparação de preços de produtos em Ivaiporã.

Este projeto foi limpo para remover React e Expo, e agora inclui uma aplicação web mobile first para administração e gerenciamento de produtos, lojas e preços.

Estrutura do repositório:
- `web/` - aplicação web mobile first em HTML, CSS e JavaScript.
- `backend/` - documentação e integração com Firebase para banco de dados e autenticação.
- `regras/` - regras e requisitos do app.
